=======
Presets
=======

Presets are ready-made rule bundles that you apply to your configuration
with one line each.

Three optional preset packages are available:

- ``flowd/phirewall-preset-owasp-crs``: OWASP Core Rule Set detection
- ``flowd/phirewall-preset-bots``: block AI crawlers, limit aggressive SEO bots
- ``flowd/phirewall-preset-bad-ips``: block known bad IP addresses

This page will be extended with usage examples for every preset.
